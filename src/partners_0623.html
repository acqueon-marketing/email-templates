New Acqueon Engagement Features

Mode-Based Dialing, Campaign Group Pacing & Enhanced Greenzone

1. Mode-Based Dialing (Contact Strategy)

The new Mode-Based Dialing feature in the Simple and Advanced Contact Strategy enables contact centers to target specific numbers more effectively.

With the Simple Contact Strategy, users can now:

  • Increase the mode retry (from a previously fixed single retry); once increased, all modes in the contact strategy get the same number of retries; each cycle is treated as complete only after all retries for all modes are exhausted

  • Change the callable times for the modes; when changed, the callable time for one mode is replicated across all the modes in the strategy

With the Advanced Contact Strategy, users can now:

  • Use the Override feature to disable select modes and allow dialing only to the selected (Overridden) modes

Overridden strategies are indicated on the listing page as Advanced (Overridden).

2. Campaign Group Pacing (Pacing Mode Enhancements)

All settings for each pacing mode ‚Äì Predictive, Progressive, Progressive IVR, and (Direct) Preview are now on the Campaign Group Page.

For ease of use, the settings are grouped by tabs based on the functionality of the pacing mode. Pacing Mode Settings include Throttling, General Settings, Answering Machine Treatment, Abandoned Call Treatment, and Contact Parameters.

This enhancement will enable our customers to use multiple campaigns or multiple lists on the same group of agents dialing with the preferred pacing mode.

3. Enhanced Greenzone

The Enhanced Greenzone ensures that contacts are delivered ONLY if all the uploaded modes for the contact are eligible to be dialed as per the CSS condition. If the contact has three modes, the application checks if all three modes of the contact are dialable before delivering it to the dialer. Contacts are not delivered if even one zone is out of the dialable time.

The Enhanced Greenzone feature also checks if any State Law is enabled.
